Variables in Js

Variables are used to store any information.


There are 3 ways to declare a Variable :-


1. Var

Var => In VAR Re-Declaration and Re-Assignement is possible.


2. let

Let => In LET Re- Declaration is not possible and Re-Assignement is possible.


3. Const

Const => In CONST Re-Declaration and Re-Assignement both are not possible.
