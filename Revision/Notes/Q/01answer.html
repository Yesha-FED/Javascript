Q1. What is JavaScript

JavaScript is a programming language that allows you to implement complex features on web pages.

It is designed for creating essential applications.


Q2. What are the ways to Apply JavaScript?

Embedding code: Adding the JavaScript code between a pair of 'script'........'script' tag.

External file: Creating a separate JavaScript file.

Inline code: Placing JavaScript code directly inside HTML tags using some special attributes.


Q3. Features of Javascript?

JavaScript is a high-level interpreted programming language.
It is the technologies of the World Wide Web, along with HTML and CSS.
It can be used to develop web applications, web pages interactive and dynamic, add features such as menus, forms, and animations.


Q4. Which company developed JavaScript?

JavaScript was created at Netscape Communications (it's a company) by Brendan Eich in the year of 1995.

Netscape and Eich designed JavaScript as a scripting language for use with the company's flagship web browser.


Q5. How many type of varibale in js?

There are three types of variable:

'var' :- In VAR Re-Declaration and Re-Assignement is possible.

'let' :- In LET Re- Declaration is not possible and Re-Assignement is possible.

'const' :- In CONST Re-Declaration and Re-Assignement both are not possible.
